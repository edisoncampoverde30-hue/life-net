# 🧠 Life Net – Mental Health Chatbot Demo

## 📌 Overview

**Life Net** is a front-end prototype of a mental health support chatbot designed to guide users through a structured support system. This project demonstrates how technology can be used to provide private, accessible, and ethical mental health assistance.

This is a **CSC 223 class project** focused on the intersection of:

* Technology
* Ethics
* Privacy
* Healthcare

## 🔁 Application Flow

The Life Net system is designed in 4 stages:

### 🔹 Stage 1 – Private Support

* Immediate chatbot-style interaction
* No personal data required
* Session is temporary (auto-delete concept)

### 🔹 Stage 2 – Profile & Intake

* User provides background and behavioral information
* Includes emotions, thought processes, and personality traits
* Data is used to personalize support

### 🔹 Stage 3 – Professional Review

* System generates a structured “review”
* Provides recommendations and suggested next steps
* Displays mental health provider options

### 🔹 Stage 4 – Appointment Booking

* User selects a provider
* Simulates scheduling a therapy session

---

## 🔐 Privacy & Security Concepts

Life Net emphasizes ethical handling of user data:

* 🔒 Encrypted session concept
* 🕒 Stage 1 data auto-deletes (simulated)
* 🧠 Profile-based data used only for support and recommendations
* ⚠️ Includes crisis awareness messaging

## 🎯 Purpose

The goal of this project is to demonstrate:

* The importance of **privacy and ethical responsibility**

## ⚠️ Disclaimer

This is a **demonstration project only**.
